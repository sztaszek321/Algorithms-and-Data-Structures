# Computer Science 2 / Algorithms

## Description
This folder contains materials for the Computer Science 2 course. It includes algorithm implementations, notes, test data, and supporting materials useful for studying and preparing for exams or quizzes.

The main purpose of this folder is to keep all course-related content in one place and make revision easier. The files stored here can be used both for understanding how individual algorithms work and for testing them on example datasets.

## Content
This folder covers algorithms and tasks related to various topics discussed during the course, including:
- calendar calculations,
- graph algorithms,
- text processing,
- classification and data analysis methods,
- voting models,
- heuristic and metaheuristic algorithms.

In addition to implementations, the folder may also include:
- short algorithm descriptions,
- sample input data,
- test cases,
- study notes,
- revision materials for quizzes or exams.

## Purpose
The main goal of this folder is to collect all important materials in one place. This makes it easier to:
- find a specific algorithm,
- compare different approaches,
- check how a program works,
- prepare for classes and exams.

## How to use
This folder should be treated as a personal study repository. It is best to keep only clear and tested implementations here, so that before an exam or quiz it is easy to quickly review the most important ideas and algorithm patterns.

The code should be simple, readable, and easy to understand. Test data should be separated from the actual implementations, and the notes should focus on the most important concepts needed for revision.

## Notes
The materials in this folder are intended for educational purposes. Their main role is to support learning, algorithm analysis, and preparation for course assessments.
